# Lab-1-Comunicaciones
Codigos de Matlab de laboratorio número 1 del ramo comunicaciones digitales

##

### 1.m
Este código al ejecutarse gráfica 5 figuras, primero una señal sinusoidal, posteriormente en la segunda muestra la comparación entre la señal sinusoidal y su PAM con muestreo natural, luego la comparación con la PAM con muestreo instantáneo y después ambas. Finalmente se gráfica la transformada de Fourier de la señal original y ambas PAM.

### 2.m
Este código gráfica 2 figuras, primero la señal sinusoidal comparada con su PAM instantánea y la PAM cuantificada. Luego gráfica el error generado por la cuantización.
