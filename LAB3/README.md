# Lab-3-Comunicaciones-Digitales
Codigos de Matlab de laboratorio número 3 del ramo comunicaciones digitales

##

### act_prev.m
Este código corresponde a las actividades previas realizadas en matlab. Al ejecutarse, grafica la señal binaria unipolar, la envolvente compleja de ASK/OOK y la transformada de Fourier de la envolvente compleja.

### act_pres.m
Este código corresponde a las actividades presenciales realizadas en matlab. Al ejecutarse, grafica la señal binaria polar, la parte real de la envolvente compleja g(t) y la transformada de Fourier de g(t).

##

Los archivos .grc corresponden a las gráficas realizadas en GNU Radio Companion
